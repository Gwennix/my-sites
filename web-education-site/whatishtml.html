WAT IS HTML?

HTML (Hyper Text Markup Language) is een programmeer taal waarmee je websites kan maken.
Het werd in 1991 bedacht en ontwikkeld door Tim Berners-Lee die tevens de eerste webbrowser ontwikkelde.
Ondertussen zitten we op versie 5 ook wel HTML5 genoemd.
Sinds het een ‘Text Markup Language’ is kun jezelf HTML schrijven in alle tekst editors maar
het programma van Microsoft Visual Studio Code is een goede aanrader.

OPZETTEN VAN EEN PAGINA

Het opzetten van een pagina begint als volgt; je hebt een DOCUMENT, waarin je eerst een ‘index.html’ file in
plaats, in die zelfde file in je VS Code tik je op je toetsenbord een ‘!’ in en klik je daarna op ‘ENTER’. Je
krijgt daarna een DOC!TYPE te zien in je file, haal dit NIET weg, want dit heb je nodig om content te creëren
in je website. 


          <!DOCTYPE html>
          <html lang="en">
          <head>
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <title>Document
          </head>
          <body>
    
          </body>
          </html>

In HTML gebruik je allemaal verschillende tags ' </>'ieder hun eigen specialiteiten met wat je vervolgens wilt
doen voor je website, zoals de 'title' tag, dit kan jij veranderen op de manier hoe je dat wilt,
voor nu staat hij opgeslagen als 'Document'.
LET OP: Als je nog verder content wilt creëren moet je dat doen IN de
body tag, want als je je website wilt zien op een LIVE SERVER, gaat het programma dat niet vinden.

Elementen en attributes

Een ‘element’ is alles tussen de tags, bijvoorbeeld:

          
            <h1> TEKST </h1>

Dus de h1 is de start van het element, de tekst is de content van het element en de /h1 is het einde van het element. Een element kan ook ‘nested’ elements in zich hebben, dat betekent dat in een element (voornamelijk html, body etc.) er een andere element komt, bijvoorbeeld: 

            
              <html>
              <body>

               <h1>TITLE</h1>
               <p> Lorem</p>

               </body>
               </html>

Je hebt ook ‘attributes’, het is extra informatie van een tag. Bijvoorbeeld in de a-tag, zit er een href attribute in: 

            
              <a href = “https://flexedu.instructure.com/” target = “_blank” > Dashboard in Canvas

‘target = “_blank”’ is niet een verplichte attribute, maar kan soms handig zijn. Het stuurt je namelijk naar de pagina in de ‘href’ in een nieuwe tab in je browser. Of misschien: 

              
                <div class=“customclassname” id=“customidname”>
                   <h2 class=“another-custom_name”> Lorem </h2>
                </div>

HEADINGS

Wanneer je een stuk tekst schrijft wil je waarschijnlijk ook een mooie titel erboven.
Daar komen heading-tags van pas!
Ze bestaan uit verschillende groottes die je per tag kan aangeven.
De tag schrijf je als <h1> en kan oplopen tot <h6>.

Meestal gebruik je de h1 eenmalig per pagina en de h2 en h3 een paar keer.

PARAGRAPHS

In HTML is de paragraaf element een p;

Voorbeeld: Paragrafen kan je gebruiken hoeveel je wilt, als het maar afgesloten is met de juiste tags. Als je in je code twee aparte paragrafen heb gezet, wordt het zo in je live server weergegeven, een line-height dat je zelf ook in CSS kan aanpassen.

LINKS

De <a> tag is een hyperlink, het stuurt je van de ene pagina naar de ander. Om dit succesvol te doen, heb je in het element <a> de ‘href’ attribuut nodig. In de ‘href’ zet je dan de link naar de andere pagina.

              
                <a href = “https://flexedu.instructure.com/”> Dashboard in Canvas< /a>

IMAGES

Met de <img> tag kan je afbeeldingen linken naar je website.
Deze tag heeft geen tweede waarmee je hem afsluit.

Een voorbeeld van hoe de code eruit ziet:
<img src="url" alt="placeholder">

LAYOUT TAGS

<header> - Definieert een koptekst voor een document of sectie.
<nav> - Definieert een reeks navigatielinks.
<section> - Definieert een sectie in een document.
<article> - Definieert een onafhankelijke, op zichzelf staande inhoud.
<aside> - Definieert inhoud naast de inhoud (zoals een zijbalk).
<footer> - Definieert een footer voor een document of sectie.
<details> - Definieert aanvullende details die de gebruiker op verzoek kan openen en sluiten.
<summary> - Definieert een kop voor het <details> element. 


          <!DOCTYPE html>
          <html lang="en">
          <head>
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <title>Document</title>
          </head>
          <body>

               <header>
                     <h1>TITLE OF YOUR CHOICE<h1/>
               </header>

               <section>
                      <nav>
                         <ul>
                           <li><a href=""></a></li>
                           <li><a href=""></a></li>
                           <li><a href=""></a></li>
                         </ul>
                      </nav>

                      <article>
                             <h2>TITLE OF YOUR CHOICE.</h2>
                             <p>CONTENT</p>
                      </article>
               </section>

          </body>
          </html>

TABLES

Een table kan je zien als een tabel, want het is letterlijk dat.
tr - /tr is table row, dus makkelijk te onthouden.
th - /th is als de bovenste rij van je tabel, de table headers, of het onderwerp van td - /td
td - /td is de cel van de table, of de content. 

          
            <table>
              <tr>
                  <th>A</th>
                  <th>B</th>
                  <th>C</th>
              </tr>
              <tr>
                  <td>1</td>
                  <td>2</td>
                  <td>3</td>
              </tr>
              <tr>
                  <td>1</td>
                  <td>2</td>
                  <td>3</td>
              </tr>
          </table>

FORMS

Veel websites hebben een systeem waarmee gebruikers berichten kunnen sturen naar de beheerders.
Dat gaat via contactformulieren.
In HTML maak je die met '<form>'.
De tekstvakjes creëer je met '<input>' met '<label>' erboven een naam te geven aan ze.
<input> vraagt ook om een type invoer die ervoor wilt gebruiken.
We gebruiken "tekst" omdat we woorden willen gebruiken in de tekstvakjes.

CLASSES

In je elementen/tags kan je ‘classes’ toevoegen, dit is als je een bepaald element apart wilt zetten van al
diezelfde tags die je gebruikt.
Dit is bijvoorbeeld handig wanneer je website gaat stylen met CSS, dat de paragraaf een ander soort styling
heeft dan de paragraaf zonder class. 


          <p>Een paragraaf zonder class.</p>
          <p class="exception__paragraph">Dit is een voorbeeld, deze paragraaf heeft een class!</p>
